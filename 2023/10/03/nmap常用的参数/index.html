nmap参数及使用方法

nmap 是一个信息搜集的一个工具，它分别有四种基本功能 端口扫描，主机探测，服务识别，系统识别。
1. 端口扫描 nmap 192.168.32.148 扫描主机的「开放端口」，在 nmap 后面直接跟主机 IP（默认扫描 1000 个端口）
https://img-blog.csdnimg.cn/a6522ae06f61470493dba868d8e0639d.png
2. 指定端口 nmap 192.168.31.148
nmap 192.168.32.148 -p 80
nmap 192.168.32.148 -p 1-80
nmap 192.168.32.148 -p 80,3389,22,21
nmap 192.168.32.148 -p 1-65535
nmap 192.168.32.148 -p- # -p- 等价于 -p 1-65535
https://img-blog.csdnimg.cn/01be50514524431a82d245163fc31dc1.png
https://img-blog.csdnimg.cn/a6522ae06f61470493dba868d8e0639d.png
打开 wireshark 单击打开，选择抓包的网卡（我这是 eth0）
https://img-blog.csdnimg.cn/45ddd7cd2f024a529e87204a2c2bf446.png
输入过滤条件 ip.addr192.168.32.148 and tcp.port80 回车
https://img-blog.csdnimg.cn/2ee885fd974740abbe0653cf58304b1f.png
这个过滤条件的意思是：过滤 IP 地址是 192.168.32.180 并且 端口是 TCP 的 80 端口。
设置完过滤条件后，我们在命令行执行扫描命令，然后查看「wireshark」中的请求包。
TCP 全连接扫描 -sT nmap -sT 192.168.32.148 -p 80
https://img-blog.csdnimg.cn/6ed4b49cd7ac4b58ad61ae61ccae5fb3.png
如果端口开放，就会进行三次握手，成功建立链接，扫描的过程中，STATE 为 open。
https://img-blog.csdnimg.cn/7c6cd564d52949929e007e3fb84686fc.png
3.SYN 半链接扫描 -sS nmap -sS 192.168.31.148
https://img-blog.csdnimg.cn/6cebf6a13aba41dfb882470f989ebbca.png
如果端口关闭，就只有一次握手，扫描结果中，STATE 字段为 closed。
https://img-blog.csdnimg.cn/808b137ea3214be19c799f8dd8aecc96.png
4. 隐蔽扫描，注意！！只适于 linux 系统。
目标主机的端口发送 TCP FIN 包 或 Xmas tree 包 或 Null 包，如果收到 RST 响应包，就判定端口关闭，否则就判定端口开放或被屏蔽（open/filtered）
nmap 127.0.0.1 -p 80 -sF Fin 扫描
nmap 127.0.0.1 -p 80 -sN Null 扫描（所有 flags 都为 0 的 TCP 包）
nmap 127.0.0.1 -p 80 -sX Xmas 扫描（flags 的 FIN、URG、PUSH 都为 1 的包）
5. 主机探测 扫描网段有哪些主机在线，-sP 参数，不扫描端口，只扫描存货的主机。
nmap -sP 192.168.31.148/24
https://img-blog.csdnimg.cn/00fa98ac789b4a12a1a5906e8808be4d.png
6. 服务识别 nmap -sV 192.168.32.148 -p 80
https://img-blog.csdnimg.cn/f421f3952afb498c8b2beaf940b3a696.png
6. 系统识别 nmap 192.168.32.148 -p 80 -O
https://img-blog.csdnimg.cn/850cebe9d9df420d876015107d4b83cf.png
7. 扫描结果导出 nmap 192.168.32.148 -p 80 -oN nmap.txt
https://img-blog.csdnimg.cn/1a1c18f7e3d54897a5b71fc9f506c352.png
8. 将扫描结果导出为 XML 格式 nmap -oX 192.168.32.148 nmap.xml
8. 进行 UDP 扫描 nmap -sU 192.168.32.148
9. 进行 RPC 扫描 nmap -sR 192.168.32.148
10. 进行滑动扫描 nmap -sW 192.168.32.148
11. 扫描速度 nmap -T1 T2 T3 T4
12. 扫描所有的 nmap -A 192.168.32.148
13. 不进行端口扫描 nmap -sn 192.168.32.148/24
14. 对目标进行不进行 dns 解析和只进行 ping 检测 nmap -n -sP 192.168.32.148/24
15 追踪每个路由节点 nmap -traveroute 192.168.32.148
重要的 SYN/ACK 开放
RST/ACK 关闭
ICMP unreachable 屏蔽
以下就是常用的参数
